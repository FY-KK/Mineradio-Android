## 使用方法

### 安装 APK 到 Android 设备

```bash
# 连接手机并启用 USB 调试
adb install -r C:\Workspace\Project-012-Mineradio-Mobile\android\app\build\outputs\apk\debug\app-debug.apk
```

### 启动后端服务（重启系统后）

```bash
# 进入后端目录
cd C:\Workspace\Project-012-Mineradio-Mobile\backend

# 使用完整路径运行 docker compose
"C:\Program Files\Docker\Docker\resources\bin\docker.exe" compose up -d --build

# 查看日志
"C:\Program Files\Docker\Docker\resources\bin\docker.exe" compose logs -f
```

后端地址: `http://localhost:3000`

### 在 App 中配置后端地址

1. 打开 Mineradio App
2. 点击顶部齿轮「设置」
3. 填入后端地址：`http://<你的IP>:3000`

## 下次启动步骤

1. 重启 Windows 系统（Docker Desktop 需要重启完成安装）
2. Docker Desktop 会自动启动
3. 运行后端服务
4. 安装/更新 APK
5. 在 App 设置中配置后端地址
